📋 Power Automate setup guide
  1. Go to make.powerautomate.com and sign in with your M365 account.
  2. Click Create → Instant cloud flow.
  3. Choose trigger: When an HTTP request is received. Click Create.
  4. In the trigger, paste this JSON schema into Request Body JSON Schema: 
    {
  "type": "object",
  "properties": {
    "date":          { "type": "string" },
    "time":          { "type": "string" },
    "name":          { "type": "string" },
    "office":        { "type": "string" },
    "notes":         { "type": "string" },
    "submittedDate": { "type": "string" },
    "submittedTime": { "type": "string" },
    "retroactive":   { "type": "string" }
  }
}
  5. Add a new step: Add a row into a table (Excel Online / Business).
  6. Select your SharePoint site, the Excel file, and the table. Map each column to the matching field from the trigger body.
  7. Save the flow. Copy the HTTP POST URL from the trigger step.
  8. Paste that URL into the Webhook URL field above and click Save.

💡 Tip: In your Excel file, create a table with columns: Date, Time, Name, Office, Notes, Submitted Date, Submitted Time, Retroactive.

Renaming or removing people/offices does not alter existing log entries.
